Section 1

System Overview

QRA is a web-based restaurant ordering system. Customers browse the menu and place orders by scanning a QR code at their table — no app download required.

How it works

📱 Scan QR Code → 🍕 Browse Menu → 🛒 Place Order → 🍳 Kitchen Cooks → ✅ Served & Paid

User Roles

👤

Customer

Restaurant guests. Browse menu, add to cart, place orders.

🛠️

Admin

Restaurant manager. Full control over menu, orders, staff, billing.

🍳

Kitchen

Kitchen staff. Kitchen display system only — update order status.

⭐

Super Admin

Platform owner. Manages all restaurants across the system.

Section 3

Customer Guide — Dine-In

Step 1 — Scan the QR Code

Each table has a unique QR code. Customers point their phone camera at it — the menu opens instantly in the browser. No app download needed.

Step 2 — Browse the Menu

  • Menu is organized by categories (Starters, Mains, Drinks, etc.)
  • Tap a category to filter items
  • Each item shows name, description, price, photo, and availability
  • Discounted items show the original price crossed out and the new price in red

Step 3 — Add Items to Cart

  1. Tap + on any item to add it to the cart

  2. Optionally tap the item to add a special note (e.g., "no onions", "extra spicy")

  3. Use + / − to adjust quantities

  4. Tap the View Cart button to review

Step 4 — Place Order

  1. Review all items, quantities, and notes in the cart

  2. Adjust or remove items if needed

  3. Tap Place Order

  4. An Order Confirmation page appears with your Order ID

Step 5 — Track Your Order

StatusMeaning
PendingReceived — waiting to be prepared
PreparingKitchen is cooking your order
ReadyReady to be served
ServedDelivered to your table
💡 You can place multiple orders during your visit. All orders are grouped into a single running bill for your table.

Section 4

Customer Guide — Takeaway

  1. Scan the Takeaway QR code displayed at the counter

  2. Browse the menu and add items to your cart (same as dine-in)

  3. Enter your name so the kitchen can call you when ready

  4. Tap Place Order — you receive a confirmation with your order ID

5.3 Menu Management

Path: Admin → Menu

Adding a Menu Item

  1. Click Add Item

  2. Fill in the details:

FieldDescription
NameItem name (e.g., "Grilled Chicken Burger")
DescriptionShort description (optional)
CategorySelect from existing categories
PriceFull price
Discount %0–100 to apply a discount (e.g., 10 = 10% off)
AvailableToggle on/off to show or hide from customers
ImageUpload a photo (JPEG/PNG, max 5 MB)
💡 Toggle Available OFF to temporarily hide an item (e.g., sold out) without deleting it.

5.4 Categories

Path: Admin → Menu (Categories panel)

ActionHow
Add categoryType the name → click Add
Rename categoryClick the edit icon → type new name → save
Delete categoryClick the delete icon → confirm
⚠️ You cannot delete a category that still has menu items. Reassign or delete those items first.

5.5 Tables & QR Codes

Path: Admin → Tables

Adding a Table

  1. Click Add Table

  2. Enter the Table Number and number of Seats

  3. Click Save — a QR code is generated automatically

Printing QR Codes

OptionHow
Print one table's QRClick the print icon on that table row
Print all QR codesClick Print All QR Codes button
Print Takeaway QRClick Print Takeaway QR at the bottom of the page
💡 Print QR codes and laminate them. Place one on each table. Customers scan to order instantly.

5.6 Bills & Payments

Path: Admin → Bills

A session tracks a group's entire visit. All orders from the same table are grouped into one session until you mark it as Paid.

Bill Flow

Customer scans QR → Session opens → Orders added → Print Bill → Mark Paid
ActionHow
View billClick the table row in Open Sessions — see all items, subtotal, charges, total
Print billClick Print Bill — opens a print-ready receipt in a new tab
Mark as paidClick Mark as Paid → confirm — session closes, table resets
View historyClick the Paid tab to see all past sessions

5.7 Users Management

Path: Admin → Users

Creating a Staff Account

  1. Click Add User

  2. Enter full name, username, password, and select a role

  3. Click Save — share credentials with the staff member

RoleAccess
AdminFull admin panel — orders, menu, tables, bills, users, settings
KitchenKitchen display only — cannot access the admin panel
⚠️You cannot delete your own account.

Section 6

Kitchen Staff Guide

The Kitchen Display System (KDS) uses a dark theme and large text for easy reading under kitchen conditions. It auto-refreshes every few seconds.

Screen Layout

Shows Pending and Preparing orders side by side. Each card shows:

  • Table number or customer name (takeaway)
  • Order type (Dine-In / Takeaway)
  • Items with quantities and special notes
  • Time since the order was placed

Processing Orders

ActionButtonResult
Start cookingStart PreparingPending → Preparing
Dish is readyReadyPreparing → Ready (disappears from KDS)
ℹ️Once you mark an order Ready, it disappears from the kitchen screen. The admin then delivers it and marks it Served.

Section 7

Super Admin Guide

After login you are taken to Admin → Restaurants, the platform-wide management panel.

Creating a New Restaurant

  1. Click Add Restaurant

  2. Enter the restaurant name, admin username, admin password, and admin full name

  3. Click Create — the restaurant and its admin account are ready

Managing Restaurants

ActionHow
Edit nameClick the edit icon next to the restaurant name
View usersExpand the restaurant row — see all staff accounts
Activate / DeactivateToggle the Active switch
Impersonate userExpand restaurant → find user → click Impersonate
ℹ️ Deactivating a restaurant blocks all logins for that restaurant's staff and stops customers from placing orders.
⚠️ Impersonate lets you log in as any user for support purposes. To return to super admin, log out and log back in with your super admin credentials.

Section 8

Push Notifications

Staff receive instant browser notifications when new orders arrive — even if the browser tab is in the background.

Enabling Notifications

  1. Log in to the admin or kitchen panel

  2. When the browser asks for notification permission, click Allow

  3. Notifications are now active on your device

What a Notification Looks Like

🍽️ New Order — Table 5
3 items · $24.50

To disable, go to your browser's site settings and revoke notification permission for this site.

Troubleshooting

Customer Issues

ProblemSolution
QR code doesn't openCheck internet connection; ask staff to verify the table is active
Menu items not showingItem may be marked unavailable — contact staff
Order placed but nothing happenedWait a moment and refresh; if nothing changes, ask staff to check
Can't see previous ordersOrder history is table-specific — scan the same table's QR code

Admin & Kitchen Issues

ProblemSolution
Can't log inCheck username/password; Caps Lock off; contact super admin to reset
Restaurant says "inactive"Super admin has suspended the restaurant — contact platform support
Orders not appearingRefresh the page; check internet connection
Kitchen display is emptyNo pending or preparing orders at this time
Image upload failsEnsure file is JPG/PNG and under 5 MB; check internet connection
Bill totals look wrongVerify service charge and tax % in Settings
Push notifications not workingCheck browser notification permissions — re-enable in browser site settings

Super Admin Issues

ProblemSolution
Can't create restaurantCheck all fields are filled; username must be unique across the system
Impersonate not workingRefresh and try again; verify the target user account exists
Backend slow to respondRender free tier sleeps after 15 min inactivity — first request takes ~30 sec to wake up
